Problem Statement

    Given an image as a vector <string>, scale it up by factor. That is, if the original image is M x N, the scaled image should be M*factor x N*factor. Each character in the input vector <string> should be represented by factor x factor of the same character in the output (see examples).

Definition

    
Class:ImageEnlarger
Method:enlarge
Parameters:vector <string>, int
Returns:vector <string>
Method signature:vector <string> enlarge(vector <string> image, int factor)
(be sure your method is public)
    

Constraints

-image will contain between 1 and 50 elements, inclusive.
-Each element of image will contain between 1 and 50 characters, inclusive.
-Each element of image will contain the same number of characters.
-Each character in image will have ASCII value between 32 and 126, inclusive.
-factor will be between 0 and 50, inclusive.
-The return will contain at most 7500 characters.
